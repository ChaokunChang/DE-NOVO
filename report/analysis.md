# DNA Concatente
## Thoughts

* 观察以下交叠区域的平均长度，也可以通过复制次数进行理论计算。
* 感觉和n-gram问题有点相像，而且这里只有四个元素，一级词表很小。$ 4_^9 < 10^6 $ 也就是说词表可以大概存序列长度小于9的所有序列，这个完全存的下，而且也不会有什么问题。如果再涨以下，到10^12，有点大，这样就有18的长度，好像其实也没长多少。
